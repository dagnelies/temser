Templo

Use it as static website generator... 
# requires python 3.5+
pip install templo
python templo -s <source-directory> -t <target-directory>
# converts all *.tmd, *.tml -> *.html
...or as dynamic template server. 
# requires python 3.5+
pip install templo
python templo -s <source-directory> -p 80
# starting as web server on port 80
  • Write using plain HTML or Markdown
  • Powerful templating using Handlebars
  • Fetch data from local json files or remote APIs
  • Fully customizable themes
